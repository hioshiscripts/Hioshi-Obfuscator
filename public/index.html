Q&A

Common questions about Hioshi Obfuscator.

What does Hioshi Obfuscator do?

It takes your Lua script and rewrites it into a version that's very hard to read, while behaving exactly the same when it runs.

Is it free to use?

Yes, the tool is free to use on the site.

What's the difference between presets?

Minify only shortens the code. Weak/Medium/Strong add increasing layers of renaming, encryption, and structural changes — stronger presets are harder to reverse but slightly heavier to run.

Will obfuscation change how my script behaves?

No. The logic stays the same — only the readability of the code changes.

Do you store my scripts?

See the Privacy & Policy page for details on what is and isn't stored.
